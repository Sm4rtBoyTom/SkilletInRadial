# Skillet in radial

Skillet in radial is a mod for [The Long Dark](https://www.thelongdark.com). 
A video game developed by [Hinterland Games](https://hinterlandgames.com/).

### Skillet in radial mod adds the skillet to the radial menu so that it can be placed the same way as cooking pot or recycled can.

### Installation

- **Install** [[ModSettings](null)]

- **Drop the** **.dll** **file into your mods folder**.
- **Enjoy**!
